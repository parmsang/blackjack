
Basic rules
---
```
Player goes bust if over 21
Picture cards are worth 10 points
Ace is worth 1 or 11
Player is dealt minimum 2 cards
Player can continue to hit until he goes bust
```

Technologies Used
---

- Ruby on Rails
- JavaScript
- JQuery
- HTML
- CSS
